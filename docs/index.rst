Welcome to the official project documentation for Phase II of the NHRP Open Data Portal. This project builds on work that was carried out by Kathmandu Living Labs between September and December 2017, to publicly release data collected during the National Housing Registration and Reconstruction Survey.

Documentation of work carried out in the first phase can be found at http://open-hrrp.readthedocs.io/
